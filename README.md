# CreativeLimit
CreativeLimit plugin for PocketMine-MP Api 5.0.0
   
## Overview  
CreativeLimit For Creative Players (No Operators)
  
## Features  
>- Permissions  
>- Cant Open Chest and etc ...
>- Op No Limit
>- No Bug
>- Limit Use And Drop Items ..
